Welcome to Geometrical Layout’s documentation!

A client for the Dowell Geometrical Layout of Big Data API. By specifying the radius, length, and width, this client help users to determine the number of circles that can be arranged in a triangular packaging, minimizing wastage of space. The coordinates of the circle can also be determine.
